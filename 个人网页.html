项目一｜专注度提升 / 外卖场景 UI 设计

独立完成从背景调查、用户调研到两轮用户测试与最终 UI 迭代的校内项目。

01
  • 以背景调查与用户研究为起点，建立中保真框架与整体信息架构。
  • 采用 Think Aloud、SUS、Post-test Interview 等方法进行多轮测试。
  • 基于亲和图（Affinity Diagram）与测试反馈进行问题归纳与版本迭代。
  • 从起始页、主界面到商家界面，持续优化新手引导、筛选效率与理解成本。
  • 最终形成完整 UI 配色体系与交互逻辑，提升易学性与可用性。

项目二｜节水主题快闪活动 UI / UX 设计

四人小组合作项目，负责研究方向、用户调研方法制定，并参与完整 UI / UX 设计，3D 建模独立完成。

02
  • 围绕节水主题进行早期调研、概念生成、原型设计与两轮用户测试。
  • 设计互动式快闪装置与信息展示逻辑，结合 LED、奖励模块与问答页面强化参与感。
  • 通过 Think Aloud、Observation、Post-test Interview、Heuristic Evaluation 等方法优化流程。
  • 迭代后的最终原型适用于快闪活动现场，兼顾互动者与围观者的观看体验。
  • 具备 UI 视觉、交互路径、实体装置与 3D 建模的复合设计能力。
